社交媒体爬虫工具 v1.1
====================

这是一个功能完整的社交媒体爬虫桌面应用程序。

✨ 核心功能:
- 爬取微博和抖音指定账号的帖子
- 📡 实时监控 - 自动轮询爬取最新帖子
- 🔍 关键词过滤 - 智能匹配关键词
- 🔔 弹窗通知 - 命中关键词自动推送
- 可视化界面展示和管理
- 配置管理系统
- 数据持久化存储
- 支持打包为Windows桌面应用

快速开始:
1. 安装依赖: pip3 install -r requirements.txt
2. 运行程序: python3 run.py (或 ./start.sh)
3. 查看文档: 使用说明.md

详细信息请查看:
- 快速开始.txt - 快速入门指南
- 使用说明.md - 完整使用文档
- 项目概览.md - 技术架构说明
