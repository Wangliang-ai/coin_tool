========================================
📖 使用流程
========================================

步骤1: 配置要监控的用户
   → 切换到"任务管理"页
   → 添加微博或抖音用户
   → 首次手动爬取（建立基线）

步骤2: 设置关键词
   → 切换到"监控管理"页
   → 添加关键词（如：新品、优惠、活动）
   → 选择匹配模式
   → 设置监控间隔（建议60秒以上）

步骤3: 启动监控
   → 点击"启动监控"按钮
   → 程序自动轮询检查
   → 发现匹配帖子立即弹窗

步骤4: 接收通知
   → 自动弹出通知窗口
   → 关键词高亮显示
   → 查看完整内容
   → 点击链接跳转原帖

⚠️ 重要提示
========================================

1. 监控间隔
   - 建议设置 60 秒以上
   - 避免过于频繁被限制

2. 关键词选择
   - 使用核心词，避免通用词
   - 可设置多个关键词
   - 定期优化关键词列表

3. 网络环境
   - 需要稳定的网络连接
   - 必要时配置代理
   - 建议在服务器运行

4. 资源管理
   - 定期清理旧数据
   - 适时重启程序
   - 关注资源占用

========================================
💡 使用技巧
========================================

技巧1: 先爬取后监控
   → 避免历史帖子重复通知

技巧2: 关键词分级
   → 高优先级：新品、发布会
   → 中优先级：优惠、促销
   → 低优先级：日常分享

技巧3: 分时段监控
   → 白天: 60秒高频监控
   → 夜间: 300秒低频监控

技巧4: 测试验证
   → 先用短间隔测试
   → 确认正常后调整
